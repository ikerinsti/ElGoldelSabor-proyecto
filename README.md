# PROYECTE 1 – RESTAURANT (APLICACIÓ WEB)

## Descripció del projecte

Aquest projecte consisteix en el desenvolupament d’una aplicació web per a la gestió bàsica d’un restaurant. L’aplicació permet als usuaris consultar la carta i realitzar comandes, i als administradors gestionar els productes, les comandes i els usuaris.

El projecte està orientat a l’aprenentatge i aplicació de conceptes de desenvolupament web amb PHP, seguint una estructura MVC (Model – View – Controller).

---

## Funcionament general

L’aplicació es divideix en tres parts principals:

1. **Part pública**: accessible per als usuaris, on es pot visualitzar la carta i realitzar comandes.
2. **Part privada (administració)**: accessible només per a administradors, des d’on es poden gestionar productes, comandes i usuaris.
3. **API**: permet l’accés a dades mitjançant peticions HTTP i retorna la informació en format JSON.

---

## Funcionalitats

### Usuari

* Visualitzar la carta de productes
* Realitzar comandes
* Iniciar i tancar sessió

### Administrador

* Crear, editar i eliminar productes
* Consultar i gestionar comandes
* Gestionar usuaris
